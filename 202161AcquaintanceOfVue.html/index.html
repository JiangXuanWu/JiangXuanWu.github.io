通过 Chocolatey 安装

Chocolatey 是一个 Windows 专用的软件包管理工具。 请按照此 说明 安装 Chocolatey 。

安装 Chocolatey 之后，你就可以在控制台执行如下命令安装 Yarn 了：

1

choco install yarn

这也会确保你安装了 Node.js 。

通过 Scoop 安装

Scoop 是一个用于 Windows 的基于命令行的安装工具。 请按照此 说明 安装 Scoop 。

Scoop 安装后，你就可以在控制台执行如下命令安装 Yarn 了：

1

scoop install yarn

如果 Node.js 未被安装，scoop 会提示你安装。 例如：

1

scoop install nodejs

注意

请将您的项目目录和 Yarn 的缓存目录 (%LocalAppData%\Yarn) 列入杀毒软件的白名单中，否则会因为每次向磁盘写入文件时而被扫描，导致安装软件包变得很慢。

通过如下命令测试 Yarn 是否安装成功：

1

yarn --version

Vue开发环境搭建

Vite 是一个 web 开发构建工具，由于其原生 ES 模块导入方式，可以实现闪电般的冷服务器启动。

通过在终端中运行以下命令，可以使用 Vite 快速构建 Vue 项目。

使用 npm：

1
2
3
4

npm init @vitejs/app <project-name>
cd <project-name>
npm install
npm run dev

或者 yarn：

1
2
3
4

yarn create @vitejs/app <project-name>
cd <project-name>
yarn		//安装环境(node_modules)
yarn dev

当你的用户名中间带有空格时（如：Mike Baker），可能导致Vite生成不成功。以下代码：

1

create-vite-app <project-name>

React开发环境搭建

1
2
3

npm init @vitejs/app --template react
yarn		//安装环境
yarn dev
